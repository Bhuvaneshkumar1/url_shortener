— Features

Everything you need in one shortener.

Built with a production-grade Flask backend, every feature was engineered for performance and reliability.

⚡

Lightning Redirect

Sub-50ms redirect latency powered by SQLite indexed queries. Every click is counted in real-time.

📊

Real-Time Analytics

Track click counts, expiry status, and a 7-day activity chart — all live from the Flask API.

🔗

Custom Aliases

Choose your own short code like localhost/r/launch. Uniqueness guaranteed by the database.

⏱️

Link Expiry Control

Set links to auto-expire after 1, 7, or 30 days. Expiry checked on every redirect.

🗄️

Persistent Storage

All links stored in SQLite via SQLAlchemy ORM. Your data survives restarts.

📱

QR Code Generation

Every shortened URL can generate a scannable QR code via the built-in QR button.
